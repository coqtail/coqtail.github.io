Definition of a newton sum

Definition Nnewton_sum n a b : nat :=
  Nfinite_sum_0_n n (fun k ⇒ (Nbinomial n k) × (a ^ k) × (b ^ (n - k))).

Newton sum theorem

Theorem Nnewton : ∀ n a b,
  (a + b) ^ n = Nnewton_sum n a b.
